  <files>entities/car/arcade_car.gd</files>
  <read_first>entities/car/arcade_car.gd</read_first>
  <action>
Convert key constants to `@export` variables so they can be tuned in Godot Editor without recompiling.

**Replace these constant declarations:**

FROM:
```gdscript
const MAX_SPEED = 35.0
const ACCELERATION = 20.0
const DECELERATION = 8.0
const BRAKE_FORCE = 25.0
const TURN_SPEED = 2.5
const MAX_TURN_RATE = 2.0
const GROUND_ALIGN_SPEED = 8.0
const GRAVITY = 25.0
```

TO:
```gdscript
@export_group("Movement")
@export var MAX_SPEED = 35.0              ## Top speed (units/second)
@export var ACCELERATION = 20.0           ## How fast speed increases
@export var DECELERATION = 8.0            ## Coasting slowdown rate
@export var BRAKE_FORCE = 25.0            ## Active braking strength
@export var REVERSE_SPEED = 15.0          ## Backward max speed

@export_group("Steering")
@export var TURN_SPEED = 2.5              ## Steering interpolation speed
@export var MAX_TURN_RATE = 2.0           ## Maximum turn rate (radians/sec)

@export_group("Ground Behavior")
@export var GROUND_ALIGN_SPEED = 8.0      ## Surface alignment smoothness
@export var GRAVITY = 25.0                ## Jump/fall acceleration
```

**Keep as const (don't export):**
```gdscript
# These should NOT be @export (fixed values for game mechanics)
const HOVER_HEIGHT = 0.1
const RAYCAST_LENGTH = 1.0
const WHEEL_RADIUS = 0.35
const WHEEL_SMOOTH_SPEED = 20.0
const REST_LENGTH = 0.6
```

**Why @export:**
- Allows tuning in Godot Editor Inspector panel without editing code
- Can test different values during gameplay (hot-reload changes)
- Grouped by category for organization
- Comments (##) appear as tooltips in editor

**Why NOT export fixed values:**
- HOVER_HEIGHT/RAYCAST_LENGTH: Tied to scene geometry (shouldn't change)
- WHEEL_RADIUS/REST_LENGTH: Match mesh dimensions (must stay consistent)
- WHEEL_SMOOTH_SPEED: Visual smoothing (not gameplay-affecting)

**Implementation note:**
After this change, can adjust values in Editor → Select Car node → Inspector → "Movement" section.
