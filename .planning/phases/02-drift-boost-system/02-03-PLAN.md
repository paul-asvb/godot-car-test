```gdscript
func _update_boosting(delta: float) -> void:
	"""
	Update boost state. Handles boost decay over time.
	Called every frame while in BOOSTING state.
	"""
	boost_timer += delta
	
	# Linear decay from initial_boost_mult to 1.0 over BOOST_DURATION
	var progress = clamp(boost_timer / BOOST_DURATION, 0.0, 1.0)
	boost_multiplier = lerp(initial_boost_mult, 1.0, progress)
	
	# Exit boost when duration expires
	if boost_timer >= BOOST_DURATION:
		_exit_boost()


func _exit_boost() -> void:
	"""
	Exit boost state and return to normal.
	Called when boost duration expires.
	"""
	current_state = State.NORMAL
	boost_timer = 0.0
	boost_multiplier = 1.0
	initial_boost_mult = 1.0
```

**Why this implementation:**
- Linear decay: Uses lerp() for smooth falloff (same pattern as Phase 1)
- Progress calculation: Normalized to 0.0-1.0 over BOOST_DURATION (1.5s)
- Timer exit: Automatically returns to NORMAL when duration reached
- State reset: Clears boost variables for next boost

**Decay behavior:**
- Start: boost_multiplier = initial_boost_mult (1.10/1.20/1.35)
- During: Linearly interpolates toward 1.0
- End: boost_multiplier = 1.0, returns to NORMAL state
- Result: Smooth speed falloff over 1.5 seconds (per CONTEXT.md)
